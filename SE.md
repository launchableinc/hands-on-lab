# (CloudBees internal) Workshop instruction guide for SE
This is a guide for the instructor on what they need to do, to run this workshop for the prospect. Before the actual workshop make sure you complete these steps.

## Setup new Launchable workspace
Go to [customer success storefront](https://cloudbees.atlassian.net/wiki/spaces/LCHCTS/pages/4448921585/Storefront+Launchable+Customer+Success), create a new workspace, then use the workspace flag app to (1) enable PTS, and (2) set the state to "HANDS_ON_LAB_V2"

## Create a forked repository of workshop
You need to fork the [`launchableinc/hands-on-lab](https://github.com/launchableinc/hands-on-lab) repository to your own account

## Create a Pull Request
Make a placeholder change in `github/workflows/pre-merge.yml` and create a new PR

## Share PR link with prospect
Instruct them: “This is the PR URL from where you can run the workshop - you will make all the edits and check the results from the same place.”
